# Learning Dashboard

Flutter-Webanwendung zur Verwaltung von Schulungskursen. Erstellt als [Code Challenge](challenge.md) für Job-Interview.

## Projektstruktur
```
lib/
├── main.dart                    # App Entry Point
├── models/
│   └── course.dart              # Course Datenmodell
├── screens/
│   ├── dashboard_screen.dart    # Hauptscreen mit Kursliste
│   └── course_detail_screen.dart # Detailansicht + Completion
├── widgets/
│   └── course_card.dart         # Wiederverwendbare Card-Komponente
└── constants/
    └── app_constants.dart       # UI-Konstanten (Padding, Fonts, Strings)

assets/
└── *.png                        # Kurs-Icons
```

## Ressourcen:
- [Flutter Dokumentation](https://docs.flutter.dev/)
- [Material Design Icons](https://fonts.google.com/icons)
